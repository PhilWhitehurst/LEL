Site specific plugin for LEL 2017 functionality

== Description ==

Provides a number of short codes, actions, and filters to delivery LEL 2017 functionality.

For example

Online tracking
Rider Area
Volunteer Area
Custom post type controls



A few notes about the sections above:

*   "Contributors" is a comma separated list of contributors to the plugin
*   "Tags" is a comma separated list of tags that apply to the plugin
*   "Requires at least" is the lowest version that the plugin will work on
*   "Tested up to" is the highest version that you've *successfully used to test the plugin*. Note that it might work on
higher versions... this is just the highest one you've verified.

 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload lel2017-site-specific to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
